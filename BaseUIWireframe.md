# Public pages

Home/Product listing → product cards (image, title, price).

Product detail → add to cart, add to wishlist.

Login / Signup (customer or vendor).

# Customer dashboard

Cart page → list of items, quantity adjust, checkout button.

Checkout page → Stripe form + confirm order.

Order history → table of past orders.

Wishlist page → list of saved products.

# Vendor dashboard

Product CRUD (table + form).

Orders (table of orders that include their products).

Analytics (just numbers first, charts later).

# Admin dashboard

Vendor approvals (list, approve/reject buttons).

Analytics (site-wide totals).
