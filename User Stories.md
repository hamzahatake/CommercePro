Phase 1 — Authentication & Accounts (Foundation)

- As a user, I can register as a customer so that I can buy products.
- As a user, I can register as a vendor so that I can sell products (pending admin approval).
- As a user, I can log in with my email & password so that I can access my account.
- As a user, I can log out so that I can secure my account.
- As a user, I can reset my password via email so that I can recover access if I forget it.
- As a user, I can update my profile info so that my account stays accurate.

Phase 2 — Admin Controls (Gatekeeper for Vendors)

- As an admin, I can view a list of all vendors so that I can manage them.
- As an admin, I can approve or reject vendor signups so that only verified vendors can sell.
- As an admin, I can activate or deactivate vendor accounts so that I control who sells on the platform.
- As a vendor, I receive an email when my signup is approved or rejected so that I know my status.

Phase 3 — Products (Core Catalog)

- As a vendor, I can register for an account and wait for approval so that I can start selling once approved. (already covered in Phases 1–2, but now vendor workflow continues)
- As a vendor, I can create, update, and delete products so that I can manage my store.
- As a vendor, I can upload product images so that my listings are appealing.
- As a vendor, I can manage product inventory (update stock) so that customers see accurate availability.
- As an admin, I can manage products (edit/remove) even if created by vendors so that I can enforce quality.

Phase 4 — Customer Shopping Flow

- As a customer, I can browse products so that I can see what’s available.
- As a customer, I can search and filter products by category so that I can quickly find what I want.
- As a customer, I can view product details (title, price, description, image, stock) so that I can make informed decisions.
- As a customer, I can add or remove products from my wishlist so that I can save items for later.
- As a customer, I can add, update, or remove products in my cart so that I can prepare my order.
- As a customer, I can view my cart total so that I know the amount before checkout.

Phase 5 — Checkout & Orders

- As a customer, I can checkout using Stripe so that I can securely pay for my order.
- As a customer, I can receive an order confirmation email so that I know my purchase was successful.
- As a customer, I can view my order history so that I can track past purchases.
- As a customer, I receive an email when my order is confirmed so that I have a receipt.

Phase 6 — Vendor Order Management

- As a vendor, I can view orders that include my products so that I can fulfill them.
- As a vendor, I can view sales analytics (e.g., monthly revenue, top products) so that I can improve my business.

Phase 7 — Admin Analytics

- As an admin, I can view site-wide analytics (total sales, top vendors/products) so that I understand platform performance.

Phase 8 — Extra/Polish

- As a user, I receive an email for password reset so that I can regain access. (supporting flow)
- As a customer, I can leave a product review so that I share feedback.
- As a customer, I can rate products so that others know product quality.
- As a customer, I see product recommendations so that I discover new items.
- As a vendor, I can categorize my products into subcategories so that customers find them more easily.
- As an admin, I can export sales reports so that I can use them externally.
